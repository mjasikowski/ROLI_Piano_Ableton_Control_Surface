Syncs the Piano scale with the one selected in Ableton Live.

### Installation Steps

1. **Locate your Ableton User Remote Scripts folder**:
   
   - **Windows**: `%USERPROFILE%\AppData\Roaming\Ableton\Live 12.x.x\Preferences\User Remote Scripts\`
   - **macOS**: `~/Music/Ableton/User Library/Remote Scripts/`
   
   Replace `12.x.x` with your Ableton Live version number.

2. **Copy the script folder**:
   
   Copy the entire `ROLI_Piano` folder into the User Remote Scripts directory.

3. **Enable the script in Ableton Live**:
   
   - Open Ableton Live
   - Go to **Preferences** (Ctrl/Cmd + ,)
   - Navigate to the **Link, Tempo & MIDI** section
   - In the **Control Surface** dropdown, select **ROLI Piano**
   - Set the **Input** to your ROLI LUMI Keys
   - Set the **Output** to your ROLI LUMI Keys
